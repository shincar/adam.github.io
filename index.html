My first game

It would be Gobblet Gobbler that I learned from a board game exhibition last year. It's advance version of tik-tok that every users hold six tokens with 3 sizes for 2 respectively. Like tik-tok you tried to make your tokens 3 in a row to win the game. But in this game you got different sizes of token that allows you to cover opponent's token with bigger size of your token.
